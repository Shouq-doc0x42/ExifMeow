# 🐾 ExifMeow
### Digital Forensics Metadata Extraction Tool

ExifMeow is a lightweight **Digital Forensics & OSINT tool** that extracts metadata from images and generates a structured investigation report.

The tool helps investigators uncover hidden information embedded inside images such as camera details, timestamps, and GPS coordinates.

---

## 🔍 Overview

In many digital forensic investigations, images contain **hidden metadata** that can reveal valuable evidence about when, where, and how a photo was taken.

**ExifMeow** simplifies this process by automatically extracting and presenting the metadata in a readable format.

This project was developed during the **Cybercrime & Advanced Digital Forensics Bootcamp at Tuwaiq Academy**.

---

## ⚙️ Features

- 🖼️ Extract EXIF metadata from images
- 📍 Detect GPS coordinates (if available)
- 📷 Display camera and device information
- ⏰ Extract image timestamps
- 📑 Generate structured forensic reports
- 🔎 Useful for OSINT image investigations
- 🐾 Lightweight and easy to use

---

## 🧰 Technologies Used

- Python
- Pillow (PIL)
- EXIF metadata libraries
- Digital Forensics principles
- OSINT techniques
